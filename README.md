# 🍬 Nassau Candy Supply Chain Intelligence Dashboard

An interactive logistics analytics and business intelligence dashboard built using Streamlit, Plotly, and Python for analysing shipment operations, sales performance, delivery efficiency, and route intelligence across the Nassau Candy distribution network.

---

# 📌 Project Overview

This dashboard transforms raw shipment and sales data into meaningful operational insights through interactive visualizations, KPI monitoring, and logistics performance analytics.

The system helps identify shipping delays, evaluate regional sales performance, analyse product profitability, and optimize supply chain operations using real-time analytics and visualization techniques.

---

# 📊 Dashboard Features

- KPI Cards for Sales, Profit, Orders, and Shipping Time
- Revenue and Profit Trend Analysis
- Regional Sales Distribution
- Product Performance Analysis
- Shipping Delay Detection
- USA Route Intelligence Map
- Fastest and Slowest Delivery State Analysis
- Factory Optimization Simulator
